# 📁 Python File Handling CRUD

A simple Python project that performs CRUD (Create, Read, Update, Delete) operations on files and folders using Python.

## 🚀 Features

- 📄 Create File
- 📖 Read File
- ✏️ Update File
- 🗑️ Delete File
- 📝 Rename File
- 📁 Create Folder
- ❌ Delete Folder

## 🛠️ Technologies Used

- Python 3
- os module
- pathlib module

## 📂 Project Structure

```
Python-File-Handling-CRUD/
│
├── Filehandling.py
├── README.md
└── .gitignore
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/raunaksharma1918-eng/Python-File-Handling-CRUD.git
```

2. Open the project folder

```bash
cd Python-File-Handling-CRUD
```

3. Run the program

```bash
python Filehandling.py
```

## 📋 Menu Options

```
0 - Exit
1 - Create File
2 - Read File
3 - Update File
4 - Delete File
5 - Rename File
6 - Create Folder
7 - Delete Folder
```
